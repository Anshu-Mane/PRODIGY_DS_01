# PRODIGY_DS_01

## 📌 Task 01: Data Visualization using Bar Chart and Histogram

### 📖 Objective
Create a bar chart and histogram to visualize the distribution of categorical and continuous variables using the Titanic dataset.

---

## 📂 Dataset

Dataset Used: Titanic Dataset

Dataset Source:
https://github.com/datasciencedojo/datasets/blob/master/titanic.csv

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- VS Code

---

## 📊 Visualizations

### 1. Gender Distribution (Bar Chart)
- Visualized the distribution of male and female passengers.
- Used the **Sex** column (Categorical Variable).

### 2. Age Distribution (Histogram)
- Visualized the distribution of passenger ages.
- Used the **Age** column (Continuous Variable).
- Missing values in the Age column were removed before plotting.

## 📈 Observations
